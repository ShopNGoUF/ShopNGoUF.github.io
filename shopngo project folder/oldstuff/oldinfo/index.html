Shop 'n' Go, vi underlättar ->din <- vardag! Delade interaktiva köpslistor, med ett simpelt knapptryck kommer du att kunna lägga till/ta bort ett objekt ifrån listan! Dela den med dina vänner, släkt och familj, samla alla perspektiv och intressen på ett och samma ställe! En simpel webbaplikation som fungerar på ett flertal enheter; telefoner, laptops och tablets! Glöm dina problem med denna prima applikation! bara mat, inget tjat!
